# PythonRealState
      
      The key of this aplication is to learn how to:
      
          - Create Models in Django
          - Create Views
          - Create superusers
          - Migrations with Python and SQL
          - Manage Django Admin
          - Base Templates
          - Static files
          - Image urls
          - Adding Google Maps
          
 # Run the server
